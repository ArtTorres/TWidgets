Message TWidgets

Displays text. Helps to displays values with consistent visual format.

The current Message TWidgets are:

  • Message
  • Marquee

Message

Displays a text message.

Layout

Supports text align, margin and position.

Events

  • OnStateChanged. Raises when the assigned text changes.

Implementation

Initialize and mount as the example below.

Source

// Setup the TWidget
var message = new Message("twidget_id")
{
    Text = "Hello World"
};

// Mount and play
TWidgetPlayer.Mount(message);

Output

Hello World

Marquee

Displays a text marquee. Each item represent a text line.

Layout

Supports text align, margin, padding, border and position.

Events

  • OnStateChanged. Raises when the assigned list of text items changes.

Implementation

Initialize and mount as the example below.

Source

// Setup the TWidget
var marquee = new Marquee("twidget_id");
marquee.Items = new string[] {
    "Welcome to TWidgets",
    "<Get Started>"
};

// Mount and play
TWidgetPlayer.Mount(marquee);

Output

┌─────────────────────────────┐
│     Welcome to TWidgets     │
│        <Get Started>        │
└─────────────────────────────┘
