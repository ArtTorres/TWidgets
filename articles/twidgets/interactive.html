Interactive TWidgets

Displays text and waits to interact with user inputs.

The current Interactive TWidgets are:

  • StopMessage
  • TextInput
  • OptionInput

StopMessage

Displays a message and stop until a key is pressed.

Layout

Supports text align, margin and position.

Events

No events trigger on this TWidget.

Press here to see interaction flows.

Validations

  • Any key input is required.

Implementation

Initialize and mount as the example below.

Source

// Setup the TWidget
var stop = new StopMessage("twidget_id");
message.Text = "-- Press Any Key to Continue --";

// Mount and play
TWidgetTWidgetPlayer.Mount(stop);

Output

-- Press Any Key to Continue --
_

*And waits for key input

Templates

No templates available for this TWidget.

TextInput

Displays a message, stops and waits for text input.

The captured value can be retrieved by the Values property.

Layout

Supports text align, margin and position.

Events

No events trigger on this TWidget.

Press here to see interaction flows.

Validations

  • Non empty inputs required.

Implementation

Initialize and mount as the example below.

Source

// Setup the TWidget
var input = new TextInput("twidget_id");
input.HeaderText = "Enter a city name:";

// Mount and play
TWidgetTWidgetPlayer.Mount(input);

// Retrieve values
foreach (var value in input.Values)
{
    Console.WriteLine("{0}: {1}", value.Key, value.Value);
}

Output

perform capture

Enter a city name:
> _

*And waits for text input

printing values

text-input.value: Madrid

OptionInput

Displays a list of text options and waits for user input.

Layout

Supports text align, margin and position.

Events

No events trigger on this TWidget.

Press here to see interaction flows.

Validations

  • Non empty inputs required.
  • Input must be a number.
  • Input number must be in range of options.

Implementation

Initialize and mount as the example below.

Source

// Setup the TWidget
var option = new OptionInput("twidget_id")
{
    InstructionsText = "Select an option >>",
    ErrorText = "Option no available, select one of the list!",
    Items = new string[] {
        "Item A",
        "Item B",
        "Item C"
    },
    NumberSeparator = ')'
};

// Mount and play
TWidgetTWidgetPlayer.Mount(option);

// Retrieve values
foreach (var value in option.Values)
{
    Console.WriteLine("{0}: {1}", value.Key, value.Value);
}

Output

perform capture

1) Item A
2) Item B
3) Item C
Select an option >> _

*And waits for text input

printing values

option-input.value: 1

No templates available for this TWidget.
