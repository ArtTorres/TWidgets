TWidget Lists

Displays a list of text items.

The current List TWidgets are:

  • BulletList
  • NumberList

BulletList

Displays a list of bulleted items.

Layout

Supports text align, margin and position.

Events

  • OnStateChanged. Raises when the assigned list of text items changes.

Implementation

Initialize and mount as the example below.

Source

// Setup the TWidget
var list = new BulletList("twidget_id");
list.Items = new string[] {
    "Uno",
    "Dos",
    "Tres"
};

// Mount and play
TWidgetPlayer.Mount(list);

Output

■ Uno
■ Dos
■ Tres

Templates

The following templates are available for this TWidget:

  • Bullet Template

OrderList

Displays a list of items with a numerical sequence.

Layout

Supports text align, margin and position.

Events

  • OnStateChanged. Raises when the assigned list of items changes.

Implementation

Initialize and mount as the example below.

Source

// Setup the TWidget
var list = new NumberList("twidget_id");
list.Items = new string[] {
    "Uno",
    "Dos",
    "Tres"
};

// Mount and play
TWidgetPlayer.Mount(list);

Output

1. Uno
2. Dos
3. Tres

Templates

No templates required for this TWidget.
