Drawing System

On TWidgets, the drawing system is inspired in the Drawing library of Microsoft .Net Framework.

The drawing workflow is as the image below:

Canvas Matrix

Canvas is a matrix with a width of Console.Width and a dynamic height.

Each time we draw a stencil element...

Layer System

You can draw as many stencils as you need. Modifying the cursor you can override the canvas matrix in a previous position. Under these circuntances a new stencil will override the values of the matrix cell as new layer above the previous stencils.

Canvas Rows and Columns

All the drawing is performed on a canvas object. The canvas have a predefined positions. X=0 and Y=0 with origin at the top-left corner.

During the drawing event you can modify manually the pointers to set the next origin for the stencils in the next position.

Source

// Custom Widget
public class NewWidget : TWidgetBase
{ 
    ...
    // On Draw Event
    public override void Draw(Graphics g)
    {
        g.Draw(new Line());
    }

    ...
}

About the Stencils

Stencils allows to draw basic items in the canvas.

Currently TWidgets have four basic stencils:

  • Line
  • Rectangle
  • Text
  • List

Mixing stencils allows to draw any image.

Drawing Lines

Draws a single line on the current position of the canvas.

By default draws a line from side to side of the screen.

Example:

Source

// Custom Widget
public class NewWidget : TWidgetBase
{ 
    ...
    // On Draw Event
    public override void Draw(Graphics g)
    {
        g.Draw(new Line());
    }

    ...
}

Draws a list of text items. Each item will be drawn below the previous one.

Alignment of Stencils

Allows to specifiy the text orientation of a text stencil.

Modify the Align property of the text stencil to change the orientation.

Example of setting up the alignment:

Source

// Custom Widget
public class NewWidget : TWidgetBase
{ 
    ...
    // On Draw Event
    public override void Draw(Graphics g)
    {
        g.Draw(new Text("Hello World"){ Align = Align.Left });
        g.Draw(new Text("Hello World"){ Align = Align.Center });
        g.Draw(new Text("Hello World"){ Align = Align.Right });
    }
    ...
}

Output [20 character width console]

Hello World
    Hello World
        Hello World

Borders of Stencils

Defines the type of border and the presentation of the stencil.

Using BorderTemplate class you can change between types of borders.

See BorderTemplate

Example of setting up a border:

Source

// Custom Widget
public class NewWidget : BoxTWidget
{ 
    ...
    // On Draw Event
    public override void Draw(Graphics g)
    {
        // Define the type of border
        var border=new Border(BorderTemplate.Solid);

        g.Draw(new Rectangle(20,3, new Margin(), border));
    }

    ...
}

Margins of Stencils

Defines the extern margin of an stencil.

Any stencil with margin values will be push on the directions of the alignment settings.

[image here]

Example of setting up a border with 1 character on top and bottom and 2 on left and right:

Source

// Custom Widget
public class NewWidget : TWidgetBase
{ 
    ...
    // On Draw Event
    public override void Draw(Graphics g)
    {
        // Define the size of margin (top, left, bottom, right)
        var margin=new Margin(1,2,1,2);

        g.Draw(new Rectangle(20,3, margin, new Border()));
    }

    ...
}
