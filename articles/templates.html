Animation Template

Provides a collection of pre-defined animations.

  • BARS
  • SOLID_BOX
  • DOT

[BARS]

Displays an animation of a moving bars. Useful in scenarios when we are showing loading processes.

Example:

Cycle 1

/ Message

Cycle 2

- Message

Cycle 3

\ Message

Cycle 4

| Message

[SOLID_BOX]

Displays an animation of an intermittent block cursor.

Example:

Cycle 1

█ Message

Cycle 2

  Message

Cycle 3

█ Message

[DOT]

Displays an animation of an intermittent point cursor.

Example:

Cycle 1

. Message

Cycle 2

  Message

Cycle 3

. Message

Border Template

Provides a collection of pre-defined borders.

  • DOTTED
  • STARS
  • RIDGE
  • FILL
  • SOLID

[DOTTED]

Displays borders as points.

Output

·······························
·     Welcome to TWidgets     ·
·        <Get Started>        ·
·······························

[STARS]

Displays borders as starts.

Output

*******************************
*     Welcome to TWidgets     *
*        <Get Started>        *
*******************************

[RIDGE]

Displays borders with ridge lines.

Output

╔═════════════════════════════╗
║     Welcome to TWidgets     ║
║        <Get Started>        ║
╚═════════════════════════════╝

[FILL]

Displays borders as a solid box.

██████Welcome to TWidgets██████
█████████<Get Started>█████████
███████████████████████████████

[SOLID]

Displays borders as a solid line.

│     Welcome to TWidgets     │
│        <Get Started>        │
└─────────────────────────────┘

Progress Bar Template

Provides a collection of pre-defined progress bars. Each one define empty and filled characters.

  • SIMPLE
  • SOLID
  • ARROW
  • INVERTED_BACKSLASH
  • BACKSLASH
  • BARS

Output

[■■■■■     ] SIMPLE
¦█████     ¦ SOLID
[>>>>>     ] ARROW
[\\\\\     ] INVERTED_BACKSLASH
[/////     ] BACKSLASH
¦|||||     ¦ BARS

Custom Templates

You can define your own templates just defining a custom array of characters in the corresponding widget.

Examples:

[Setting custom bullet in BulletList]

A custom bullet support any character.

Source

// Setup the TWidget
var list = new BulletList("twidget_id");

list.Items = new string[] {
    "Uno",
    "Dos",
    "Tres"
};
list.Bullet='+';

// Mount and play
TWidgetPlayer.Mount(list);

Output

+ Uno
+ Dos
+ Tres

[Setting custom ProgressBar in ProgressBar widget]

A custom array must have 4 characters.

Source

// Setup the TWidget
var progress = new ProgressBar("twidget_id");
progress.Template= new char[]{'@', '#', '+', '@'};

// Mount and play
TWidgetPlayer.Mount(progress);

// Execute 5 Cycles
for (int i = 0; i < 5; i++)
{
    progress.PerformStep();
    Thread.Sleep(200);
}

Output

@++++++++++##########@

[Setting custom Border in Marquee widget]

A custom array must have 9 characters.

Source

// Setup the TWidget
var marquee = new Marquee("twidget_id");

marquee.Items = new string[] {
    "Welcome to TWidgets",
    "<Get Started>"
};

marquee.Border.Template = new char[]{
    '+', '+', '+',
    '+', ' ', '+',
    '+', '+', '+'
};

// Mount and play
TWidgetPlayer.Mount(marquee);

Output

+++++++++++++++++++++++++++++++
+     Welcome to TWidgets     +
+        <Get Started>        +
+++++++++++++++++++++++++++++++

[Setting a custom Animation in ProgressChar widget]

A custom array if characters of any size can be defined as template.

Source

// Setup the TWidget
var progress = new ProgressChar("twidget_id");
progress.Text = "Start Message";
progress.Template = new char[]{'@','#','%'};

// Mount and play
TWidgetPlayer.Mount(progress);

// Execute 4 Cycles
for (int i = 0; i < 4;)
{
    progress.Text = $"Message {++i:00}";
    Thread.Sleep(500);
}

Output

Cycle 1

@ Message 01

Cycle 2

# Message 02

Cycle 3

% Message 03

Cycle 4

@ Message 04
